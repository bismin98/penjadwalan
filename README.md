# Jadwal Kegiatan Walikota Balikpapan

Website jadwal kegiatan walikota Balikpapan dengan dua jalur login: panel admin dan pengguna.

## Menjalankan Lokal

```bash
npm run dev
```

## Halaman Utama

- Halaman awal: pilihan login admin dan pengguna
- Login admin: `/login/admin`
- Login pengguna: `/login/user`

## Catatan

- Form login masih berupa tampilan, belum terhubung ke backend.
